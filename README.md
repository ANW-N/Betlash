# 📄 Betlash Pro

Zamonaviy va premium dizaynga ega bo'lgan kitob, qo'llanma va hujjatlarni chop etish uchun varaqlarni avtomatik hisoblovchi web-ilova.

## ✨ Xususiyatlari

* 🖨️ Ikki tomonlama chop etish rejimi
* 📚 Kitobcha (Booklet / A5) rejimi
* 🔄 Old va orqa tomonlar uchun inversiya funksiyasi
* 🌙 Dark Mode va ☀️ Light Mode
* 📊 Jonli statistika paneli
* 📋 Bir bosishda nusxalash
* 🧊 Glassmorphism Premium UI
* 📱 Mobil qurilmalar uchun moslashuvchan dizayn
* ⚡ Tezkor hisoblash algoritmi
* 💾 Mavzu tanlovini LocalStorage orqali saqlash

---

## 🚀 Ishlatish

1. **Boshlanish** va **Tugash** betlarini kiriting.
2. Chop etish rejimini tanlang:

   * Ikki tomonlama
   * Kitobcha (A5)
3. **"Varaqlarni Hisoblash"** tugmasini bosing.
4. Natijalar:

   * O'ng (Oldi) tomon
   * Chap (Orqa) tomon
5. Kerak bo'lsa inversiya funksiyasidan foydalaning.
6. Natijani nusxalab printer dasturiga joylashtiring.
